# Stroud Football Skills

This repository hosts a **simple website for young football players** to share the skills they are learning.

The site is built as a **static website** and is hosted using **GitHub Pages**.

The goal of the project is to create a **safe, simple football learning site for children**.

---

# Website

Once GitHub Pages is enabled, the site will be available at:

https://stroud-football.github.io/stroud-football-skills/

---

# Technology

This website intentionally uses **very simple technology**.

The site contains:

- HTML
- CSS
- optional images

There is:

- no backend
- no database
- no build tools
- no frameworks

This keeps the site simple and easy to maintain.

---

# Purpose

The website shares things like:

- football skills
- practice ideas
- favourite players
- things learned from matches

It is intended for **young football players in the Stroud area**.

---

# Safety Rules

Because this site involves children, strict safety rules apply.

The website must NEVER include:

- surnames
- full names
- school names
- exact location details
- addresses
- contact details
- email addresses
- phone numbers
- team schedules
- training times
- match times
- fixture lists

Allowed information includes:

- first names only
- general references to football practice
- general location references such as "Stroud"

The goal is to **protect the privacy and safety of children**.

---

# Contributing

When editing the site:

1. Keep pages simple
2. Avoid complex code
3. Ensure content is child friendly
4. Follow all safety rules

If content includes personal or location information it must be removed.

---

# Deployment

The site is deployed automatically using **GitHub Pages**.

Publishing steps:

1. Commit changes to the repository
2. GitHub Pages automatically builds the site
3. The site updates within a few minutes
